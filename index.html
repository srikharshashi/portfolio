Projects I'm proud of

Trash Detection and Reporting using YOLOv5 and Drone Kit

  • The Trash Detection and Reporting project aims to automate the evaluation of sanitation and cleaning in large areas without manual inspection.
  • Aerial Images captured via a drone are then inferred using a custom trained YOLOv5 model
  • When the threshold is exceeded, authorities receive alerts through a ServiceNow Incident.
  • The images are stored in Amazon S3 with references in the database for easy retrieval.
  • The project includes a web app for viewing the images, and a mobile app called Drone Command to control the drone's missions.
  • Additionally, a Trash Detection Client performs GPU-enabled deep-learning inference updates the database with the results.
  • A backend server with MongoDB and Express.js supports data management and websocket based drone-control alogrithm
  • All the hosting is done on AWS using a basic EC2 instance and an S3 bucket with MongoDB Atlas for DB Hosting.

Intelligent Crime Reporting and Intervention System

  • The system allows citizens to anonymously report crimes through a mobile app, providing media evidence and exact location.
  • The Patrol App enables authorities to respond quickly, with a maximum of five cases per patrol authority scheduled using a queue based on priority.
  • The backend employs ML models for face detection and NLP crime categorization calculating a crime score for case prioritization.
  • An Admin App provides live case updates, location-wise statistics, and a centralized view for higher authorities.
  • The system incentivizes reporters with crypto tokens using ERC-20 based smart contracts
